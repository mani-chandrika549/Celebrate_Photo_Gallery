## Project Overview
This project is a **React Photo Gallery application** built as part of the Celebrare frontend internship assignment.

The application fetches photos from the **Picsum Photos API** and displays them in a responsive grid layout.

Users can:
- View photos from the API
- Search photos by author name
- Mark photos as favorites
- Persist favorites after page refresh using localStorage

## Features
- Fetch photos from API
- Responsive photo grid
- Real-time search filter
- Favorites system using `useReducer`
- Favorites saved in `localStorage`
- Performance optimization using `useMemo` and `useCallback`

## Tech Stack
- React
- JavaScript
- Tailwind CSS
- Vite
- Picsum Photos API
